# XJU-SE-Open（新疆大学软件工程学习资料库）

> 面向新疆大学软件工程专业（及相关专业）同学的学习资料与实践仓库：课程笔记、资料索引、复习要点、练习题整理（合规）、课程/自学项目与工具脚本等。

大家好，我是新疆大学软件工程专业大二学生 **曹大地**。我把自己��学习过程中的资料整理成一个开源仓库，包含但不限于：

- 课程笔记与知识点总结  
- 教材/参考资料的书目信息与获取途径（尽量提供官方链接）  
- 复习要点、练习题与解答整理（合规前提下）  
- 课程/自学的小项目与代码实践、实验报告  
- 学习工具、环境配置、效率脚本与模板  
- 学习路线、方法与经验总结  

如果这个仓库对你有帮助，欢迎点个 **Star**；也非常欢迎通过 **Issue / PR** 一起完善。

## 仓库结构

- **Notes/**：课程笔记与知识点总结（按课程/章节整理）
- **Books/**：教材与参考资料索引（书名 + 版本 + 官方链接/出版社信息等）
- **Exams/**：复习题、练习题、题型整理与解答（避免上传不合规内容）
- **Projects/**：课程/自学项目、Demo、实验代码与报告
- **Tools/**：学习工具、环境配置、效率脚本、常用模板
- **Roadmap/**：学习路线、学习方法、经验总结

## 如何使用

### 1）在线浏览（推荐）
- 直接在 GitHub 上按目录查看内容
- 需要检索时可用 GitHub 搜索：课程名 / 关键词 / 文件名

### 2）下载 ZIP
仓库页面点击：**Code → Download ZIP**

### 3）Git 克隆
```bash
git clone https://github.com/cdd0329/learn.git
```

## 贡献指南

欢迎提交（PR / Issue）：
- 更清晰的笔记结构与排版（目录、标题层级、重点标注等）
- 更正错误与补充内容（能附参考来源更好）
- 新的项目/实验/工具脚本
- 课程资料索引与学习路线（含踩坑记录/经验总结）

### 提交建议（推荐遵守）
- **命名清晰**：例如 `课程名-章节-主题.md` 或 `课程名/章节/主题.md`
- **尽量使用 Markdown**：便于阅读与检索
- **尽量可复现**：项目类内容建议包含运行方式、依赖、示例输入输出等
- **不要提交不合规材料**：见下方合规说明

## 合规说明

本仓库尊重知识产权与学校相关规定：

- **不上传**明确受版权保护且未获授权的教材扫描版/盗版资源  
- **不上传**可能涉及泄题、违规传播的内容  
- 若你认为仓库内容侵权或不合规，请通过 **Issue** 联系我，我会及时处理与下架  

> 建议：`Books/` 目录尽量以“书目信息 + 版本 + 官方链接/出版社信息”为主，避免直接上传侵权文件。
